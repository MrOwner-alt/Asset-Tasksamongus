Important Notices

We recommend keeping all the files in the same directory to make the game load faster. The best option is to keep all the files in the same drive.

Do not change the names of any files, or the launcher will not work correctly. File names are crucial for the proper functioning of the game launcher. Please ensure that the names remain exactly as they are when downloaded.

It is important to download all the files listed below to make the game work properly. Missing files can cause the game to malfunction or not start at all.
